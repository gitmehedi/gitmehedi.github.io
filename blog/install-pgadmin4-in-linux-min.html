Install pgAdmin 4 on Linux Mint

Published on September 8, 2019 5 min read
1. Open terminal and type 
$ wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -
2. Open Software Sources and click on “Additional repositories” and paste the following. 
$ deb http://apt.postgresql.org/pub/repos/apt/ bionic-pgdg main
3. it should look like this. 
$ sudo apt-get -y update
Press “OK” and that will automatically update
4. Now open terminal and type the following. 
                    
$ sudo apt update
$ sudo apt install pgadmin4
